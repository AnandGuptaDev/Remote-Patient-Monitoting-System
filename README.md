# Remote Patient Monitoring System

An IoT-based healthcare monitoring system designed to collect and display real-time patient vital parameters using multiple sensors.

## Features

- Heart Rate Monitoring
- Blood Oxygen (SpO2) Measurement
- Body Temperature Monitoring
- Motion/Fall Detection
- Real-time LCD Display
- Continuous Sensor Data Acquisition

## Hardware Used

- ESP-32 WROOM 
- MAX30102 Pulse Oximeter Sensor
- LM35 Temperature Sensor
- MPU6050 Accelerometer & Gyroscope
- 16x2 LCD Display

## Technologies Used

- Arduino C/C++
- I2C Communication
- Sensor Data Processing
- Embedded Systems
